## Spam Email Classifier (Machine Learning)

This project is a simple spam email classifier built using Python and machine learning techniques.

## Features

- Classifies emails as spam or not spam
- Uses Naive Bayes machine learning algorithm
- Converts text into numerical data using CountVectorizer

## Technologies

- Python
- Machine Learning
- scikit-learn
- NLP

## Example

Input:
Free prize waiting for you

Output:
Spam Email
